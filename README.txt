# Ridgeland JROTC Website

## Editing the Site
- Replace placeholder images in `images/`
- Edit text in each team page for real details
- Add more `<div class="team-card">` blocks in team grids for additional members
- Update waiver PDF in `files/` if needed

## Deployment
1. Make changes locally
2. Commit and push to GitHub
3. GitHub Pages will auto-update the live site
